## 開始

#### version-01-Nav-basic  使用方法: 

將 version-01-Nav-basic 內所有檔案
複製 ＝> 取代 react-template 內相同的資料(請注意各個檔案的位置)
```
npm install 
npm start
```
註: 關鍵字 tailwindcss postcss react-fontawesome postcss-cli
